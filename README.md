# Simple flask API

Python 3.9 or later
Flask==3.1.0

## config environment

Virtualenv (recommended)

### crete a virtual env

```bash
python -m venv .venv
```

Activate the environment

linux:

```bash
source venv/bin/activate
```

Windows:

```bash
.venv\Scripts\activate
```

### install requirements

```bash
pip install -r requirements.txt
```

### run app

```bash
python run.py
```
